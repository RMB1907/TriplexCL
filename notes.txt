Operating System: Windows 11 (64-bit)
Python version: 3.12.6
JupyterLab version: 4.5.0

create a virtual environment (jupyter kernel):
1. go to project root
2. python3 -m venv .venv
3. source .venv/bin/activate
4. pip install --upgrade pip
5. pip install -r requirements.txt
6. pip install ipykernel
7. python -m ipykernel install --user --name=triplex --display-name "Python (triplex)"

now, use the "Python (triplex)" kernel to run the notebooks present in this repository.
